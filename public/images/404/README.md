# 404 Page Images

Place the following images in this directory (`/public/images/404/`):

## Required Images

1. **not-found.jpg** (1200x675px recommended)
   - Hero image at the top of the 404 page
   - Should be friendly and welcoming, not scary or negative
   - Could show:
     - A lost/confused but friendly character
     - A map or navigation theme
     - Books or learning materials scattered
     - A cozy, welcoming scene that fits the brand
   - Full-width banner image

## Design Notes

- Keep the image light and friendly to match the brand's tender, supportive approach
- Avoid dark or negative imagery
- Consider incorporating elements related to learning, books, or navigation
- The image should feel welcoming and guide users back to the learning journey
