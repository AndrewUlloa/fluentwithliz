## Image Guidelines

- **Format**: JPG or WebP (WebP preferred for better compression)
- **Aspect Ratios**: Maintain the recommended aspect ratios for best display
- **Optimization**: Images will be automatically optimized by Next.js Image component
- **Alt Text**: Already included in the component, but ensure images match the descriptions
- **File Names**: Use exact file names as listed above (case-sensitive)

## Image Placeholders

Until images are added, the page will show loading skeletons. The images will gracefully handle missing files with error states.
